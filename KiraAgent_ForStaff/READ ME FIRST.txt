KIRA AGENT — QUICK START FOR STAFF
===================================

WHAT THIS IS
This program connects the SQL Accounting on THIS computer to Kira Cloud.
Once set up, approved batches from the Kira console appear inside SQL
Accounting automatically — nobody has to key them in.

BEFORE YOU START
- Run this on the computer where SQL Accounting is INSTALLED.
- Have the SQL Accounting username/password for each company ready.
- This is a TRIAL — everything starts in "dry run" mode, meaning it will
  practice the whole process but will NOT write anything into your real
  SQL data until we turn that off together.

HOW TO RUN IT
1. Keep "KiraAgent.exe" and the ".env" file in the SAME folder (don't
   separate them).
2. Double-click "KiraAgent.exe".
3. The FIRST time, it will ask you a few questions on screen:
     - Name this PC (anything, e.g. "office-pc-1")
     - It will scan and show you the SQL company files it found
     - Pick a company number
     - Enter that company's SQL Accounting username & password
     - It will try to read the company's name automatically
     - Press Enter to accept the suggested name (or type an existing
       one if this company was already set up before)
   That's it — the company now appears in Kira Cloud automatically.
4. Repeat step 3 (run it again, or answer more prompts) for every company
   this computer handles.
5. After setup, just leave the window open — it checks for approved work
   every 30 seconds. Closing the window stops it.

A LOG FILE
Everything it does is saved next to the program in "kira_agent.log" —
send us this file if anything looks wrong.

QUESTIONS / SOMETHING LOOKS WRONG
Stop and message us with a screenshot before doing anything else —
don't worry, nothing can be posted into real SQL data while it's in
dry-run mode.
